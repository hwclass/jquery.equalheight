jquery.equalheight
==================

Responsive Equal Height Plugin for jQuery


usage (native)
==================

$(window).load(function() {
	$('.elementsToBeFixedByHeight').equalHeight();
});


usage (jQuery)
==================

$(doeument).ready(function() {
	$('.elementsToBeFixedByHeight').equalHeight();
});


usage (resize - must)
==================
$(window).resize(function(){
	$('.elementsToBeFixedByHeight').equalHeight();
});
