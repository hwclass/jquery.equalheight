## jquery.equalheight
==================

Responsive Equal Height Plugin for jQuery


### usage (native)
==================
<pre lang="javascript">
<code>
$(window).load(function() {
	$('.elementsToBeFixedByHeight').equalHeight();
});
</code>
</pre>


### usage (jQuery)
==================
<pre lang="javascript">
<code>
$(document).ready(function() {
	$('.elementsToBeFixedByHeight').equalHeight();
});
</code>
</pre>
